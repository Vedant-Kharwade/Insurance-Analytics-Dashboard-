# 🏆 Prism Insurance Analytics Dashboard (Power BI Project)

An end-to-end data analytics project that analyzes **insurance policies, claims, customer feedback, and financial performance**, transforming raw insurance data into actionable business insights.

Built using **Power BI and Excel**, this dashboard highlights trends in **premium, coverage, claims, policy status, and customer sentiment**.

## 📊 Key Insights

### 💰 Financial Insights
- **Travel Insurance** generates the highest premium (~2.5M)
- **Health & Auto policies** are mid-performing segments
- **Life & Home insurance** show comparatively lower premium contribution

---

### 📌 Claims Analysis
- **Rejected Claims:** ~4.4K (highest)
- **Settled Claims:** ~3.4K
- **Pending Claims:** ~2.3K

👉 Indicates **high rejection ratio**, potential issue in claim validation or policy clarity

---

### 👥 Customer Demographics
- Gender distribution is balanced:
  - **Male:** 5000
  - **Female:** 5000

- **Adults generate the highest claim amount (~10M)**  
- **Elder group contributes moderate claims (~6M)**  
- **Young adults have minimal claims**

---

### 📈 Policy Status
- **Active Policies:** 56.9%
- **Inactive Policies:** 43.1%

👉 Good retention but still scope for improvement

---

### 💬 Customer Feedback Insights
- Majority feedback categorized as:
  - **Improvement (57%)**
  - **Excellent (26%)**
  - **Good (14%)**

- Word cloud highlights:
  - Positive: *“Great service”, “Easy claim”, “Helpful staff”*
  - Negative: *“Website issues”, “Policy confusion”, “High premium”*

👉 Key improvement areas:
- Website performance
- Policy clarity
- Pricing transparency

## 🔍 Workflow

### 1. Data Collection
- Imported insurance dataset (policy + claims + customer feedback)

### 2. Data Cleaning & Transformation
- Removed duplicates
- Handled missing values
- Standardized date formats
- Created calculated columns (Age Group, Claim Status, etc.)

### 3. Data Modeling
- Built relationships between:
  - Policies
  - Customers
  - Claims

### 4. Visualization (Power BI)
- Designed dark-themed interactive dashboard
- Used:
  - KPI Cards
  - Bar Charts
  - Donut Charts
  - Area Charts
  - Word Cloud
  - Tables

### 5. Insight Generation
- Identified high claim rejection rates
- Found top-performing policy types
- Analyzed customer sentiment patterns
